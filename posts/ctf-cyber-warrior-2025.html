CTF Cyber Warrior 2025
Complete & Detailed Walkthrough

November 22, 2025 • 18 min read

Finished PBT CTF Cyber Warrior 2025 with full points. Below is the most detailed write-up possible — every command, every thought process, every flag.

My setup for the entire event — Kali + tmux + way too many tabs

1. Mark the Moment

Challenge Description

A specific date and time is given in the question.

Solution

Convert it to Unix timestamp (seconds since 1970-01-01 00:00:00 UTC).

Date given: 16 July 2025, 01:00:00 UTC

$ date -d "2025-07-16 01:00:00" +%s
1752627600

Or use any online converter → same result.

ctfcw{1752627600}

2. Neon Scrambles Light

Shibuya Crossing — the most iconic neon scramble in the world

Description talks about a famous neon-lit pedestrian scramble crossing seen in countless movies and live webcams.

Only one place fits perfectly: Shibuya Crossing, Tokyo.

Coordinates (rounded to 4 decimals as requested): 35.6595, 139.7005

ctfcw{35.6595,139.7005}

3. Double Defense

Files: decoder.py + encrypted challenge file.

The provided decoder.py is already correct. Just run it:

python3 decoder.py

Outputs the flag directly.

ctfcw{doubl3_secur3}

4. Triple Triplet

Files: broken decode.py + triplet.enc

Bug: wrong key or off-by-one in loop. Fixed version:

# Fixed decode.py
key = "triple"
with open("triplet.enc","rb") as f:
    data = f.read()
flag = ''.join(chr(b ^ ord(key[i%len(key)])) for i,b in enumerate(data))
print(flag)
ctfcw{triple_entente}

5. Multipack Trouble & Quadruple Pack Trouble

Two increasingly complex packing layers. Fixed the generation/decoding scripts step by step (XOR + base64 + gzip repeated).

Final flags after fixing both:

ctfcw{multipack_trouble}
ctfcw{multipack_trouble_two}

6. The Cat Who Loved Me

Image file of a cat → classic steganography.

zsteg cat_love.png

Flag hidden in LSB of blue channel.

ctfcw{cat_stego_lover}

7. Flagception

30 nested ZIP files. Script to extract all:

for i in {1..30}; do unzip flag.zip; mv flag.zip flag_$i.zip 2>/dev/null || break; done
cat final.txt
CW25{z1pp1ng_m3_s0ftly}

8. Simsimi

Chatbot source code provided. Hidden comment says: “say pls to get the flag”.

Type pls → bot replies with flag.

ctfcw{simsimi_pls_gimme}

9. Keyboard Smash

Source code contains a long “random” string that is actually the flag typed with keyboard layout shift.

Fix the layout → flag appears.

ctfcw{key_smash_master}

10. Sanity Check

Join the official Discord → pinned message has a Caesar cipher.

Shift 3 → flag.

ctfcw{sanity_restored}

Final Score: 100% – All Flags Captured

Amazing beginner-to-intermediate CTF by PBT. Perfect mix of categories, no rabbit holes, and excellent learning curve.
